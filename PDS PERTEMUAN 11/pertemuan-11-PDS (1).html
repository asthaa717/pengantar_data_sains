1. Data dan Preview Data¶

Dataset mpg berisi informasi mengenai karakteristik mobil, termasuk konsumsi bahan bakar (mpg) dan tenaga mesin (horsepower). Data dibersihkan terlebih dahulu dari nilai kosong (missing values) agar siap digunakan untuk analisis lebih lanjut. Di sini saya memilih bahan bakar (mpg) sebagai variabel dependen (Y) dan tenaga mesin (horsepower) sebagai variabel independen (X).

3. Uji Asumsi Normalitas¶

Uji normalitas digunakan untuk memverifikasi apakah sisa (residual) dari model menyebar normal. Uji ini penting dalam regresi. Jika nilai p > 0.05, maka residual dianggap berdistribusi normal.

In [5]:
residuals = model.resid

# Shapiro-Wilk test
stat, p = stats.shapiro(residuals)
print(f'Shapiro-Wilk Test: stat={stat}, p={p}')

# Q-Q plot
sm.qqplot(residuals, line='s')
plt.title("Q-Q Plot Residual")
plt.show()

4. Uji Asumsi Heteroskedastisitas¶

Uji ini digunakan untuk mengetahui apakah varians residual bersifat konstan (homoskedastis). Jika nilai p > 0.05, maka tidak ada heteroskedastisitas, sehingga asumsi regresi terpenuhi.

In [6]:
bp_test = het_breuschpagan(residuals, X)

6. Uji Asumsi Autokorelasi¶

Autokorelasi mengukur apakah residual pada model saling berkorelasi antar observasi. Ini sangat penting terutama dalam data time series. Nilai statistik Durbin-Watson yang mendekati 2 menunjukkan bahwa tidak ada autokorelasi dalam residual.

7. Permodelan Regresi¶

Membangun model regresi linear dengan satu prediktor yaitu horsepower, dan satu variabel respon yaitu mpg. Model regresi ini dirancang untuk memprediksi mpg berdasarkan nilai horsepower. Fungsi add_constant() digunakan untuk menambahkan intercept dalam model.

In [4]:
X = sm.add_constant(df['horsepower'])  
y = df['mpg']                          
model = sm.OLS(y, X).fit()

print(model.summary())

8. R-square¶

R-squared manual dihitung sebagai proporsi dari total variasi dalam data yang dijelaskan oleh model. Nilai ini harus identik atau sangat mendekati nilai yang diberikan oleh model.rsquared.

In [12]:
y_pred = model.predict(X)
residuals = y - y_pred
SSres = np.sum(residuals**2)
SStot = np.sum((y - np.mean(y))**2)
R_squared_manual = 1 - (SSres / SStot)
print(f"R-squared (manual): {R_squared_manual:.4f}")

R-squared (manual): 0.6059

9. Interpretasi Model¶

​ persamaan regresinya: y = 39.9359−0.1578x

Intercept (39.94): Ketika horsepower = 0, maka nilai prediksi mpg adalah 39.94.

Koefisien horsepower (-0.157): Setiap penambahan satu unit horsepower, mpg akan menurun sebesar 0.157, yang menunjukkan hubungan negatif.

R-squared (0.60): Sekitar 60% variasi mpg dapat dijelaskan oleh variabel horsepower
